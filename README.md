<h1>📋 Sistema de Gerenciamento de Tarefas</h1>
<br>
📌 Sobre o Projeto

O Sistema de Gerenciamento de Tarefas é uma aplicação desenvolvida para permitir o cadastro, organização, atualização e remoção de tarefas de forma simples e eficiente.

O projeto foi construído utilizando Node.js no back-end, MySQL para persistência dos dados e HTML/CSS na interface, aplicando conceitos de desenvolvimento full stack e operações CRUD (Create, Read, Update e Delete).

<br>

<h3>⚙️ Funcionalidades</h3>

O sistema permite:

Criar novas tarefas <br>
Listar tarefas cadastradas <br>
Atualizar tarefas existentes <br>
Excluir tarefas <br>
Alterar descrição de tarefas <br>

<br>

🔌 Configuração do Ambiente <br>
<br>
Instalar dependências <br> 
 - npm install <br>
 <br>
Configurar variáveis de ambiente <br>

Arquivo .env <br>

PORT=3333 <br>

DB_HOST=localhost <br>
DB_USER=root <br>
DB_PASSWORD=sua_senha <br>
DB_NAME=task_manager <br>
Executar projeto <br>
<br>
npm run dev <br>
ou
node index.js
